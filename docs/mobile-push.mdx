---
title: Implement Mobile Push Notifications
subtitle: Learn how to send mobile push notifications from your mobile app
---

MagicBell can help you implement mobile push notifications in four steps.

1. [Setup iOS (APNS) notifications](/docs/mobile-push/ios)
2. [Setup Android (Firebase) notifications](/docs/mobile-push/android)
3. Configure your mobile app.
4. Register device tokens with MagicBell.

The last two steps depend on the framework you are using to build the app. Please find specific instructions for

- [React Native](/docs/react-native/implement-mobile-push-notifications)
