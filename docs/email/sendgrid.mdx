## Sending SendGrid emails from MagicBell's API

### Initial setup

- Sign up for SendGrid and obtain a SendGrid API key
- Email your API key to hello@magicbell.com

### Sending the email

<Note>
  Refer to our documentation to view all the parameters for overrides.
</Note>

Now it's simple! Just include the snippet below in the API:

```shell hideHeader
"overrides": {
    "providers": {
        "sendgrid": {
                "from": {"email": "enter_email_address"}
        }
    }
}
```

## Using SendGrid email templates

After creating a template in SendGrid, make sure you have the template ID. Then add the template_id attribute to overrides, and any custom data in your SendGrid template. Below is an example showing how to add a users name to each template.

```shell hideHeader
"overrides": {
    "providers": {
        "sendgrid": {
            "from": {"email": "dan@magicbell.io"},
            "personalizations": 
            [
                {
                    "to": [{"email": "dan@magicbell.io"}],
                    "dynamic_template_data":{
                        "company": "MB",
                        "Sender_Name": "Dan"
                    }
                }
            ],
            "template_id":"d-1a686dcde88644659ba2a90435280a78"
        }
    }
}
```
