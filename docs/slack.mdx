## Deliver notifications to your Slack workspace

To have Slack configured for your MagicBell project, follow these steps:

- Sign up for Slack workspace.
- Create a Slack application and obtain an oauth key.
- Send us your oauth key.
- Verify the slack channel in MagicBell is enabled

## Sign up for a Slack Workspace
Go to https://slack.com/ and Create a new workspace.

## Create a slack application and obtain an oath key
Head over to https://api.slack.com/apps and click on the "Create New App" button
<img
  className="full-mobile"
  src="/docs/uploads/yourApps.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>
Name your slack app to something that makes sense to you, eg MagicBell, select the workspace you'd like to install it to and cleck on Create.
<img
  className="full-mobile"
  src="/docs/uploads/renameAppAndSelectWorkspace.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>
Under the Basic Information section, click on the "Bots" section. Next we'll assign the required scopes to your bot token to ensure messages reach the correct recipients.
<img
  className="full-mobile"
  src="/docs/uploads/basicInformationSection.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>
Click on the "Review Scopes to Add" button.
<img
  className="full-mobile"
  src="/docs/uploads/reviewScopes.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>
Under the "Bot Token Scopes" add the following scopes:
<img
  className="full-mobile"
  src="/docs/uploads/botTokenScopes.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>
The only thing remaining is to install the app to your workspace:
<img
  className="full-mobile"
  src="/docs/uploads/installToWorkspace.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>
Click "Allow"
<img
  className="full-mobile"
  src="/docs/uploads/allowSlackApp.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>
This takes you to the OAuth & Permissions page. From here copy the Bot User OAuth Token from the  OAuth Tokens for Your Workspace section and send it over.
<img
  className="full-mobile"
  src="/docs/uploads/botUserOAuthToken.png"
  style={{ width: '75%', height: 'intrinsic' }}
/>

## Verify the slack channel in MagicBell is enabled

Go to `https://app.magicbell.com/projects/project_id/channels` and ensure that the slack channel is enabled.

## Send a notification to a slack user/channel

Once your account is set up, MagicBell will automatically deliver slack
notifications through your Slack app, **no additional work is needed**.

<Tabs>

```json title=PAYLOAD hideHeader noTopBorderRadius
{
  "notification": {
    "title": "Welcome to Example Inc",
    "recipients": [{
      "email": "dan@example.com",
      "externalId": "AB1234BCA",
      "slackChannelId": "AB1234BCA"
    }],
    "custom_attributes": {
      "name": "Dan",
      "company": {
        "name": "Example Inc"
      }
    }
  }
}
```

```curl title=cURL hideHeader noTopBorderRadius
curl https://api.magicbell.com/notifications \
  --request POST \
  --header 'X-MAGICBELL-API-KEY: MAGICBELL_API_KEY' \
  --header 'X-MAGICBELL-API-SECRET: MAGICBELL_API_SECRET' \
  --data '{
    "notification": {
      "title": "Welcome to Example Inc",
      "recipients": [{
        "email": "dan@example.com",
        "externalId": "AB1234BCA",
        "slackChannelId": "AB1234BCA"
      }],
      "custom_attributes": {
        "name": "Dan",
        "company": {
          "name": "Example Inc"
        }
      }
    }
  }'
```
